# Movies app

A flask application where you can save all the movies you've seen, with own summary and review. All public information come from the TMDB api.

---

# Setup

## Flask

### Installation

You need to setup all the flask environment to run the app.
Visit this [link](https://flask.palletsprojects.com/en/stable/installation/) to learn how to setup flask.

### Database

Run `python3 init_db.py` to create the database.
If you need to update the database manually, modify `update_db.py` in the spot designated and then run `python3 update_db.py` to run your sql command.

![Database schema](/db_schema.png)
Database schema

## Dependencies

### Dotenv

Install dotenv : `pip install python-dotenv`
[More information](https://pypi.org/project/python-dotenv/)

Create a [tmdb account](https://www.themoviedb.org/signup), then create your developer profile and [redeem your api key](https://www.themoviedb.org/settings/api).
Copy the `.env.example` file as `.env` and paste you tmdb api key in the field.

### Requests

Install requests : `pip install requests`
[More information](https://pypi.org/project/requests/)
